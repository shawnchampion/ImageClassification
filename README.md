# 卷积神经网络-图像分类
该项目为2019年1月校内深度学习实训项目

## 文件信息
cats_and_dogs_small.rar 3000张猫狗图片用于训练  
cats_and_dogs_small_1.h5 初次训练后的神经网络模型  
cats_and_dogs_small_2.h5 数据增强后的再次训练的模型  
小数据集的卷积神经网络.ipynb 项目主流程，包含代码及结果
